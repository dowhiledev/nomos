What's New in v0.2.4

Enhanced Testing, Tool Configuration & External Tools

This release brings powerful testing capabilities, streamlined tool management, and external tool support for CrewAI and LangChain.

Smart Testing

New smart_assert and ScenarioRunner for LLM-based response validation.

Config-Based Tools

Define tools directly in your agent config file instead of relying on CLI flags. Better organization and deployment.

YAML Test Configs

Load test configurations from YAML files with nomos test command.

CrewAI & LangChain Tools

Seamlessly integrate ready-made tools from CrewAI or LangChain for powerful agent capabilities.

Built for Production
AI Agents

From rapid prototyping to enterprise deployment. NOMOS scales with your needs.

Step-based Architecture

Define agent behavior as sequences of steps. Each step controls what tools are available and how decisions flow.

Multi-LLM Support

OpenAI, Mistral, Gemini, Ollama, HuggingFace, and more. Switch providers with configuration changes, not code rewrites.

Precise Tool Control

Control exactly which tools your agent can access at each step. No more unwanted function calls.

Configuration-Driven

YAML or Python configuration. Now in v0.2.4: Integrated tool configuration with CrewAI & LangChain tool support.

Session Management

Built-in persistence with Redis or PostgreSQL. Stateful conversations that survive restarts.

Deploy Anywhere

Docker, Kubernetes, AWS, Azure, GCP. Production-ready deployment options.
