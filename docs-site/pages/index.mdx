# Nomos

An open-source, configurable multi-step agent framework for building advanced LLM-powered assistants.

[Nomos on GitHub](https://github.com/dowhiledev/nomos)

## Features

- Step based agent flows
- Persona driven conversation
- Tool integration with auto documentation
- YAML or Python configuration
- Visual Flow Builder
- Multiple LLM providers
- Advanced error handling and tracing
